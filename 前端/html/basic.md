要掌握HTML，核心是理解它是**用于构建网页结构的标记语言**，而非编程语言。它通过标签描述网页中的文本、图片、链接等元素的位置和意义，最终由浏览器解析并呈现。

### 一、HTML基础核心概念
这是入门的基石，必须先理解以下3个核心要素。

#### 1. HTML文档结构（固定模板）
任何HTML文件都遵循固定的骨架结构，新建HTML文件时可直接复制使用。
```html
<!DOCTYPE html>  <!-- 声明文档类型为HTML5，必须放在第一行 -->
<html lang="zh-CN">  <!-- 根标签，lang属性定义页面语言（zh-CN为中文） -->
  <head>  <!-- 头部：存放页面元信息，不直接显示在页面上 -->
    <meta charset="UTF-8">  <!-- 关键！定义字符编码为UTF-8，避免中文乱码 -->
    <title>页面标题</title>  <!-- 浏览器标签栏显示的标题 -->
    <link rel="stylesheet" href="style.css">  <!-- 引入外部CSS样式（后续关联） -->
  </head>
  <body>  <!-- 主体：页面所有可见内容都放在这里（文字、图片、按钮等） -->
    你好，这是我的第一个HTML页面！
  </body>
</html>
```

#### 2. 标签（Tag）的规则
标签是HTML的“积木”，所有内容都通过标签包裹，分为3类：
- **双标签**：有开始和结束标签，内容放在中间，格式为 `<标签名>内容</标签名>`。  
  例：`<p>这是一段文字</p>`（段落标签）、`<h1>这是一级标题</h1>`（一级标题）。
- **单标签**：只有开始标签，无内容，需在末尾加 `/` 闭合（HTML5中可省略，但建议写）。  
  例：`<img src="图片路径" alt="图片加载失败提示">`（图片标签）、`<br/>`（换行标签）。
- **嵌套规则**：标签可嵌套，但不能交叉嵌套。  
  ✅ 正确：`<div><p>正确嵌套</p></div>`  
  ❌ 错误：`<div><p>交叉嵌套</div></p>`

#### 3. 属性（Attribute）的作用
属性是标签的“附加信息”，写在开始标签内，格式为 `属性名="属性值"`，用于修改标签的功能或样式。
- 例1：`<a href="https://www.baidu.com" target="_blank">跳转到百度</a>`  
  - `href`：定义链接的目标地址（核心属性）。  
  - `target="_blank"`：定义点击链接后在新窗口打开（`_self` 为当前窗口，默认值）。
- 例2：`<img src="logo.png" alt="网站logo" width="200" height="100">`  
  - `src`：定义图片的路径（核心属性，必须有）。  
  - `alt`：图片加载失败时显示的文字（提升 accessibility，建议有）。  
  - `width/height`：定义图片的宽高（单位默认是像素）。


### 二、常用HTML标签分类（必学）
按功能可分为6大类，掌握这些标签就能搭建大部分网页的基础结构。

#### 1. 文本结构标签（定义文字层级）
用于区分标题、段落、强调等文本，影响页面的语义和默认排版。
| 标签       | 功能说明                  | 特点                          |
|------------|---------------------------|-------------------------------|
| `<h1>-<h6>` | 一级到六级标题            | h1字体最大（一个页面建议1个），h6最小 |
| `<p>`      | 段落                      | 段落间自动换行，有默认间距    |
| `<strong>` | 加粗强调（语义：重要内容）| 浏览器默认显示为粗体          |
| `<em>`     | 斜体强调（语义：强调语气）| 浏览器默认显示为斜体          |
| `<br/>`    | 强制换行                  | 单标签，用于段落内的换行      |
| `<hr/>`    | 水平线                    | 单标签，用于分隔不同内容区块  |

#### 2. 容器标签（用于布局分组）
本身无默认样式，主要作用是“包裹其他标签”，方便后续用CSS控制样式或用JS操作。
- `<div>`：块级容器，独占一行（默认宽度100%，高度由内容决定）。  
  用途：划分大的布局区块，如“头部”“导航栏”“主体内容”“底部”。
- `<span>`：行内容器，不独占一行（宽度和高度由内容决定）。  
  用途：包裹行内的小部分内容，如“段落中的某个关键词”，用于单独修改样式。

#### 3. 链接与媒体标签（插入外部资源）
- **链接标签 `<a>`**：核心属性是 `href`，用于跳转页面或锚点。  
  例1：跳转到外部网站：`<a href="https://www.bilibili.com" target="_blank">B站</a>`  
  例2：跳转到当前页面的锚点（需先定义锚点）：  
  ```html
  <a href="#section1">跳转到第一部分</a>  <!-- 点击跳转 -->
  <div id="section1">这是第一部分内容</div>  <!-- 锚点：id值与href的#后内容一致 -->
  ```

- **图片标签 `<img>`**：核心属性是 `src`（图片路径），常见路径分为2种：  
  - 相对路径：图片文件与HTML文件在同一项目文件夹内，如 `src="images/logo.png"`（images是子文件夹，logo.png在其中）。  
  - 绝对路径：完整的网络地址，如 `src="https://img01.360buyimg.com/n1/jfs/t1/198812/33/25449/123456/6478abcde123/abcdef123456789.jpg"`。

- **视频标签 `<video>`**：用于嵌入视频，支持MP4、WebM等格式。  
  例：`<video src="video/movie.mp4" controls width="600">您的浏览器不支持视频播放</video>`  
  - `controls`：显示播放控件（播放/暂停、音量、进度条）。  
  - 标签内的文字：浏览器不支持该标签时显示的备用文本。

#### 4. 列表标签（有序/无序列表）
用于展示多条并列的内容，如“菜单”“步骤说明”“商品列表”。
- **无序列表 `<ul>` + `<li>`**：列表项前默认是圆点（可通过CSS修改为方块、空心圆等）。  
  例：
  ```html
  <ul>
    <li>苹果</li>
    <li>香蕉</li>
    <li>橙子</li>
  </ul>
  ```

- **有序列表 `<ol>` + `<li>`**：列表项前默认是数字（可通过CSS修改为字母、罗马数字等）。  
  例：
  ```html
  <ol>
    <li>打开浏览器</li>
    <li>输入百度网址</li>
    <li>搜索“HTML教程”</li>
  </ol>
  ```

#### 5. 表格标签（展示结构化数据）
用于展示行和列的二维数据，如“学生成绩表”“商品价格表”，核心是 `<table>` + `<tr>` + `<td>`。
```html
<table border="1">  <!-- border="1"：显示表格边框（默认无边框） -->
  <tr>  <!-- table row：表格的一行 -->
    <th>姓名</th>  <!-- table header：表头单元格，默认居中加粗 -->
    <th>年龄</th>
    <th>班级</th>
  </tr>
  <tr>
    <td>张三</td>  <!-- table data：普通数据单元格 -->
    <td>18</td>
    <td>高一(1)班</td>
  </tr>
  <tr>
    <td>李四</td>
    <td>17</td>
    <td>高一(2)班</td>
  </tr>
</table>
```

#### 6. 表单标签（用于用户输入）
核心是 `<form>` 包裹输入控件，用于收集用户信息（如“登录”“注册”“搜索”），常见输入控件如下：
| 标签/属性         | 功能说明                  | 示例代码                          |
|-------------------|---------------------------|-----------------------------------|
| `<input type="text">` | 单行文本输入框            | `<input type="text" placeholder="请输入用户名">`（placeholder是提示文字） |
| `<input type="password">` | 密码输入框（输入内容隐藏） | `<input type="password" placeholder="请输入密码">` |
| `<input type="button">` | 普通按钮                  | `<input type="button" value="点击我">`（value是按钮文字） |
| `<input type="submit">` | 提交按钮（提交表单数据）  | `<input type="submit" value="登录">` |
| `<select>` + `<option>` | 下拉选择框                | `<select><option value="1">男</option><option value="2">女</option></select>` |
| `<textarea>`      | 多行文本输入框            | `<textarea rows="5" cols="30" placeholder="请输入备注">`（rows/cols定义宽高） |

完整表单示例：
```html
<form action="login.php" method="post">  <!-- action：数据提交的目标地址；method：提交方式（post/get） -->
  <div>
    <label for="username">用户名：</label>  <!-- label关联输入框，点击文字也能聚焦输入框 -->
    <input type="text" id="username" name="username" placeholder="请输入用户名">  <!-- id与label的for一致；name是数据提交的键名 -->
  </div>
  <div>
    <label for="password">密码：</label>
    <input type="password" id="password" name="password" placeholder="请输入密码">
  </div>
  <input type="submit" value="登录">
</form>
```


### 三、HTML5新增特性（提升体验）
HTML5在原有基础上增加了更具语义化的标签和功能，让代码更易读、更贴合实际布局。

#### 1. 语义化布局标签（替代部分`<div>`）
这些标签功能与`<div>`类似，但有明确的“语义”，方便浏览器、搜索引擎理解页面结构（对SEO友好）。
| 标签       | 语义含义                  | 用途                          |
|------------|---------------------------|-------------------------------|
| `<header>` | 页面或区块的“头部”        | 存放网站logo、标题、导航栏等  |
| `<nav>`    | “导航栏”                  | 存放页面跳转链接（如首页、关于我们、联系我们） |
| `<main>`   | 页面的“主体内容”          | 存放页面核心内容（一个页面建议只有1个） |
| `<section>` | 页面中的“独立区块”        | 如“新闻列表区块”“商品推荐区块” |
| `<article>` | “独立的文章内容”          | 如博客文章、新闻详情、商品介绍 |
| `<footer>` | 页面或区块的“底部”        | 存放版权信息、联系方式、备案号等 |

#### 2. 新增表单类型（简化输入）
`<input>` 的 `type` 属性新增多种类型，减少JS验证代码，提升用户体验：
- `type="email"`：邮箱输入框，提交时浏览器会自动验证格式（如是否包含 `@`）。  
- `type="tel"`：电话输入框，在移动端会弹出数字键盘。  
- `type="number"`：数字输入框，默认显示加减按钮，限制输入非数字内容。  
- `type="date"`：日期选择框，浏览器会弹出日历控件（无需自己写日历）。


### 四、学习建议与实践步骤
掌握HTML的关键是“理解+多写”，按以下步骤练习效率更高：

1. **搭建环境（5分钟搞定）**  
   - 无需复杂软件，推荐2个工具：  
     - 编辑器：VS Code（免费，下载后安装“Open in Browser”插件，可直接在浏览器预览HTML文件）。  
     - 预览：写完代码后，右键点击HTML文件 → 选择“Open in Default Browser”，即可看到效果。

2. **从简单到复杂的实践任务**  
   - 任务1：写一个“个人介绍页”，包含标题（`<h1>`）、段落（`<p>`）、个人照片（`<img>`）、跳转链接（`<a>`）。  
   - 任务2：在任务1基础上，用`<ul>`做一个“我的爱好”列表，用`<div>`划分“头部”“主体”“底部”。  
   - 任务3：写一个“登录表单”，包含用户名、密码输入框和提交按钮（用`<form>`相关标签）。  
   - 任务4：用HTML5语义化标签（`<header>` `<nav>` `<main>` `<footer>`）重构任务2的页面。

3. **查错与学习资源**  
   - 常见错误：中文乱码（检查是否加了 `<meta charset="UTF-8">`）、图片不显示（检查`src`路径是否正确）、标签未闭合（检查双标签是否有`</标签名>`）。  
   - 参考文档：MDN Web Docs（https://developer.mozilla.org/zh-CN/docs/Web/HTML），权威且免费，每个标签都有详细说明和示例。


### 五、注意事项
- HTML不负责“样式”：HTML只定义“结构”，如“这是一个标题”“这是一张图片”；页面的颜色、字体、间距等样式，需要用**CSS**控制。  
- 语义化优先：写标签时优先考虑“语义”而非“样式”。例如，需要标题就用`<h1>-<h6>`，而非用`<div style="font-size:24px;font-weight:bold">`，语义化有利于SEO和代码维护。  
- 兼容性：HTML5的部分标签（如`<header>` `<nav>`）在非常老旧的浏览器（如IE8及以下）不支持，但现在主流浏览器（Chrome、Edge、Firefox、Safari）都已完全支持，无需过度担心。

在 VS Code 中快速搭建 HTML 代码基本框架有多种便捷方法，以下是常用的几种：

### 1. 使用 Emmet 缩写（最推荐）
VS Code 内置了 Emmet 语法支持，直接输入缩写后按 **Tab** 键即可生成完整框架：
- 输入 `!` 或 `html:5`，然后按 **Tab** 键，会自动生成 HTML5 标准模板：
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
```

### 2. 通过代码片段（Snippets）
VS Code 自带 HTML 代码片段，输入关键词后选择提示：
- 输入 `html`，在智能提示中选择 `html:5` 并回车，效果同上。
- 若需自定义模板，可通过「文件 > 首选项 > 配置用户代码片段 > html.json」添加自定义片段。

### 3. 使用快捷命令
- 新建 `.html` 文件后，按下 `Ctrl+Shift+P`（Windows/Linux）或 `Cmd+Shift+P`（Mac）打开命令面板。
- 输入并选择「HTML:5」或「Emmet: Expand Abbreviation」，同样能快速生成基础框架。

这些方法都能帮助你在几秒内完成 HTML 基本结构的搭建，其中 Emmet 缩写 `! + Tab` 是最快捷高效的方式。
