PowerShell 提供了丰富的命令（cmdlet）用于管理系统、处理数据和自动化任务。以下是按功能分类的常见常见命令及其用法详解：
# 一、文件与目录操作
## 1. Get-ChildItem（别名：dir、ls）
- **功能**：列出目录内容（文件和子目录）
- **常用参数**：
    - Path：指定路径（默认当前目录）
    - Recurse：递归列出所有子目录
    - Filter：按名称筛选（如 *.txt）
    - File/-Directory：只显示文件 / 目录
- **示例：**
```shell
Get-ChildItem -Path C:\Docs -Filter *.pdf -Recurse  # 递归查找所有PDF文件
dir -File  # 只显示当前目录的文件
```
## 2. New-Item（别名：ni）
- **功能**：创建文件或目录
- **常用参数**：
    - Path：路径
    - ItemType：指定类型（File 或 Directory）
- **示例：**
```shell
New-Item -Path D:\Logs -ItemType Directory  # 创建目录
ni report.txt -ItemType File  # 创建空文件
```
## 3. Copy-Item（别名：cp、copy）
- **功能**：复制文件 / 目录
**示例：**
```shell
Copy-Item C:\data.txt D:\backup\  # 复制文件到目录
cp -Path *.docx -Destination E:\docs -Recurse  # 递归复制所有docx文件
```
## 4. Move-Item（别名：mv、move）
- **功能**：移动或重命名文件 / 目录
- **示例：**
```shell
Move-Item oldname.txt newname.txt  # 重命名文件
mv D:\temp\* C:\archive\  # 移动目录中所有文件
```
## 5. Remove-Item（别名：rm、del）
- **功能**：删除文件 / 目录
- **常用参数**：
    - Force：强制删除（包括只读文件）
    - Recurse：删除目录及所有内容
- **示例：**
```shell
Remove-Item junk.txt -Force  # 强制删除文件
rm -Path tempdir -Recurse  # 删除目录及内容
```
## 6. Get-Content（别名：gc、type）
- **功能**：读取文件内容
- **示例：**
```shell
Get-Content config.ini  # 读取文件所有行
gc log.txt -Head 5  # 读取前5行
```
## 7. Set-Content（别名：sc）
- **功能**：写入内容到文件（覆盖原有内容）
- **示例：**
```shell
"Hello World" | Set-Content output.txt  # 写入字符串到文件
```
# 二、系统管理
## 1. Get-Process（别名：ps）
- **功能**：查看运行中的进程
- **示例：**
```shell
Get-Process  # 列出所有进程
ps -Name chrome  # 只显示Chrome进程
```
## 2. Stop-Process（别名：kill）
- **功能**：终止进程
- **示例：**
```shell
Stop-Process -Name notepad  # 终止所有记事本进程
kill -Id 1234  # 通过进程ID终止
```
## 3. Get-Service
- **功能：**查看系统服务状态
- **示例：**
```shell
Get-Service -Name *sql*  # 查找名称含sql的服务
Get-Service | Where-Object Status -eq Running  # 只显示运行中的服务
```
## 4. Start-Service/Stop-Service
- **功能**：启动 / 停止服务
- **示例：**
```shell
Start-Service -Name wuauserv  # 启动Windows更新服务
Stop-Service -Name spooler  # 停止打印服务
```
## 5. Get-ComputerInfo
- **功能**：获取系统详细信息
- **示例：**
```shell
Get-ComputerInfo | Select-Object OSName, OSVersion  # 查看操作系统信息
```
# 三、数据处理与筛选
## 1. Where-Object（别名：?）
- **功能**：按条件筛选数据
- **示例：**
```shell
Get-ChildItem | Where-Object Length -gt 100000  # 筛选大于100KB的文件
Get-Process | ? CPU -gt 50  # 筛选CPU使用率超50%的进程
```

## 2. Select-Object（别名：select）
- **功能**：选择对象的属性或限制结果数量
- **示例：**
```shell
Get-Service | Select-Object Name, Status  # 只显示服务名称和状态
Get-ChildItem | select -First 5  # 只显示前5项
```
## 3. Sort-Object（别名：sort）
- **功能**：对结果排序
- **示例：**
```shell
Get-ChildItem | Sort-Object Length -Descending  # 按文件大小降序排列
```

## 4. ForEach-Object（别名：%）
- **功能**：循环处理每个对象
- **示例：**
```shell
Get-ChildItem *.txt | ForEach-Object { $_.Length }  # 显示所有txt文件大小
"a", "b", "c" | % { "$_123" }  # 为每个字符串添加后缀
```
# 四、网络操作
## 1. Test-Connection（别名：ping）
- **功能**：测试网络连接（类似 ping）
- **示例：
```shell
Test-Connection -ComputerName google.com -Count 3  #  ping谷歌3次
```

## 2. Invoke-WebRequest（别名：iwr）
- **功能**：发送 HTTP 请求（下载文件、网页内容等）
- **示例：**
```shell
Invoke-WebRequest -Uri "https://example.com" -OutFile "example.html"  # 下载网页
iwr -Uri "https://api.example.com/data"  # 获取API数据
```

## 3. Get-NetIPAddress
- **功能**：查看网络接口 IP 地址
- **示例：**
```shell
Get-NetIPAddress | Select-Object InterfaceAlias, IPAddress  # 显示网卡和IP
```
# 五、变量与环境
## 1. 变量操作
- **定义变量**：$name = "value"
- **查看所有变量**：Get-Variable
- **删除变量**：Remove-Variable name
## 2. Get-ChildItem Env:
- **功能**：查看环境变量
- **示例：**
```shell
$env:PATH  # 查看PATH环境变量
$env:USERNAME  # 查看当前用户名
```
# 六、其他常用命令
- **Clear-Host（别名：cls）**：清空控制台
- **Get-History（别名：h）**：查看命令历史
- **Invoke-History 5（别名：r 5）**：重复执行历史中第 5 条命令
- **Export-Csv**：将数据导出为 CSV 文件
```shell
Get-Process | Select-Object Name, Id | Export-Csv processes.csv
```

- **Import-Csv**：导入 CSV 文件数据
**使用技巧**
1. **别名查询**：用 Get-Alias 查看所有别名（如 Get-Alias dir 可查看 dir 对应的实际命令）。
2. **管道操作**：通过 | 组合命令（如 Get-Process | Where-Object ... | Select-Object ...）。
3. **帮助查询**：忘记用法时，用 Get-Help 命令名 -Examples 查看示例。
掌握这些命令可以应对大部分日常操作，复杂任务可通过组合命令或编写脚本实现。
